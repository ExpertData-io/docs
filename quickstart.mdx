## Step 1: Clone Repositories

```bash
# Create project directory
mkdir expertengine && cd expertengine

# Clone all repositories
git clone https://github.com/ExpertData-io/expertdata-backend.git
git clone https://github.com/ExpertData-io/expertengine-frontend-react.git
git clone https://github.com/ExpertData-io/expertengine-executor.git
git clone https://github.com/ExpertData-io/expertengine-infra.git
```

## Step 2: Start Services

```bash
cd expertengine-infra/local

# Start all services
make up

# Wait for services to be healthy (30-60 seconds)
make status
```

<Note>
  The first startup may take 2-3 minutes as Docker pulls images and initializes
  databases.
</Note>

## Step 3: Access the Application

Once services are running, open your browser:

| Service      | URL                                               | Description     |
| ------------ | ------------------------------------------------- | --------------- |
| **Frontend** | [localhost:3000](http://localhost:3000)           | Web application |
| **API Docs** | [localhost:8030/docs](http://localhost:8030/docs) | Swagger UI      |
| **MinIO**    | [localhost:9001](http://localhost:9001)           | Object storage  |

## Step 4: Create Your Account

<Steps>
  <Step title="Open the App">
    Navigate to [localhost:3000](http://localhost:3000)
  </Step>
  <Step title="Sign Up">Click **Sign Up** and enter your details</Step>
  <Step title="Verify Email">
    For local development, email verification is disabled
  </Step>
</Steps>

## Step 5: Create Your First Connector

<Tabs>
  <Tab title="Using the UI">
    1. Go to **Dashboard** → **Connectors** 2. Click **+ New Connector** 3.
    Select **PostgreSQL** 4. Enter connection details: - **Name**: `Demo
    Database` - **Host**: `postgres` - **Port**: `5432` - **Database**:
    `expertengine` - **Username**: `postgres` - **Password**: `postgres` 5.
    Click **Test Connection** ✅ 6. Click **Save**
  </Tab>
  <Tab title="Using Genie">
    Open **Genie** and type: ``` Create a PostgreSQL connector called "Demo
    Database" connecting to host postgres, port 5432, database expertengine,
    username postgres, password postgres ```
  </Tab>
</Tabs>

## Step 6: Ask Genie

Open **Explorer** → **Genie** and try these queries:

```
Show me all the tables in the Demo Database
```

```
Count how many rows are in each table
```

```
Query the first 10 rows from the users table and show as a chart
```

<Check>
  **Congratulations!** You've successfully set up ExpertEngine and run your
  first analysis.
</Check>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Services won't start">
    ```bash
    # Check Docker is running
    docker ps

    # Restart services
    make down && make up
    ```

  </Accordion>
  <Accordion title="Database connection failed">
    ```bash
    # Check PostgreSQL logs
    docker compose logs postgres
    ```
  </Accordion>
  <Accordion title="Frontend shows blank page">
    Clear browser cache and reload, or check frontend logs:
    ```bash
    docker compose logs frontend
    ```
  </Accordion>
</AccordionGroup>
