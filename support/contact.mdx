## Before You Contact Us

Please have the following ready:

- Your account email

- Steps to reproduce (if applicable)
- Any error messages

## Common Issues

| Issue                     | Quick Fix                             |
| ------------------------- | ------------------------------------- |
| Can't connect to database | Check credentials and firewall rules  |
| Query timeout             | Optimize query or check database load |
| Pipeline failed           | Check task logs for error details     |
| Genie not responding      | Refresh the page and try again        |
